    Class MonoSingleton<T>

    Extend to create singletons from MonoBehaviours that can be lazy loaded from resources using ResourceSingletonAttribute or instantiated at runtime. Lazy loaded and runtime instantiated singletons will be moved to DontDestroyOnLoad.
