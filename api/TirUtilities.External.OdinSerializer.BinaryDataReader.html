    ExitArray()

    Exits the closest array. This method will keep skipping entries using SkipEntry() until an EndOfArray is reached, or the end of the stream is reached.

    This call MUST have been preceded by a corresponding call to EnterArray(out Int64).

    This call will change the values of the IsInArrayNode, CurrentNodeName, CurrentNodeId and CurrentNodeDepth to the correct values for the node that was prior to the exited array node.
